Layout

Site Width Container

This entire page content is wrapped in a .site-width-container class that provides responsive width constraints and proper margins.

Container Properties:
  • Left/right margins: var(--layout-right-left-margin)
  • Max width on large screens: var(--site-width)
  • Automatically centered on screens wider than var(--site-width)

Floating Action Buttons (FAB)

Floating action buttons represent the primary action in a screen. They appear above other UI elements and provide prominent access to key actions.

Standard FAB

Circular buttons with a single icon, perfect for the most important action on a screen.

FAB Colors

<button class="btn btn--fab btn--primary" aria-label="Add item">
  <svg class="btn__icon" width="24" height="24">
    <!-- Plus icon -->
  </svg>
</button>

FAB Sizes

<!-- Small FAB -->
<button class="btn btn--fab btn--primary btn--small" aria-label="Small FAB">
  <svg class="btn__icon" width="20" height="20"><!-- Icon --></svg>
</button>

<!-- Standard FAB (default) -->
<button class="btn btn--fab btn--primary" aria-label="Standard FAB">
  <svg class="btn__icon" width="24" height="24"><!-- Icon --></svg>
</button>

Extended FAB

Extended FABs include both an icon and text label, providing more context for the action.

<button class="btn btn--fab-extended btn--primary">
  <svg class="btn__icon" width="24" height="24">
    <!-- Icon -->
  </svg>
  <span class="btn__text">Create</span>
</button>
